## 如何使用此文件

每次发布新版本时：
1. 在文件顶部添加新版本的发布说明
2. 更新 `package.json` 的 `version` 字段
3. 运行构建命令：`npm run build:win:local`
4. 发布说明会自动包含在更新提示中

### 发布说明模板

```markdown
## vX.Y.Z (YYYY-MM-DD)

### 新增功能
- ✨ 功能描述

### 优化改进
- 🎨 改进描述
- ⚡ 性能优化
- 💾 其他优化

### 问题修复
- 🐛 修复的问题

### 已知问题
- ⚠️ 待解决的问题
```
